# Anforderungen Event-ERP

## Zielgruppe

Eventbranche: Veranstaltungsplaner, Event-Agenturen, Location-Betreiber

## Kernmodule (Vorschlag)

### 1. Kundenmanagement (CRM)
- [ ] Kundenstamm mit Kontaktdaten
- [ ] Vertragsmanagement
- [ ] Kommunikationshistorie

### 2. Event-Planung
- [ ] Event-Anlagen mit Details (Datum, Ort, Typ)
- [ ] Timeline / Ablaufplanung
- [ ] Ressourcen-Zuweisung (Personal, Equipment)

### 3. Personal-Management
- [ ] Mitarbeiterstamm
- [ ] Verfügbarkeiten verwalten
- [ ] Einsatzplanung

### 4. Inventar / Equipment
- [ ] Artikelverwaltung
- [ ] Verfügbarkeitscheck
- [ ] Wartungshistorie

### 5. Buchhaltung
- [ ] Angebote erstellen
- [ ] Rechnungen generieren
- [ ] Zahlungsverfolgung

### 6. Reporting
- [ ] Umsatzübersichten
- [ ] Auslastungsreports
- [ ] Kunden-Analysen
