# EVO_MT - Eventorganisation Management Tool

Ein ERP-System für die Eventbranche als Web-Anwendung.

## Tech-Stack

| Layer | Technologie |
|-------|-------------|
| Frontend | React + TypeScript |
| Backend | Node.js + Express |
| Datenbank | PostgreSQL |
| Deployment | Docker |

## Projektstruktur

```
EVO_MT/
├── docs/           # Dokumentation & Anforderungen
├── src/            # Quellcode
│   ├── frontend/   # React App
│   └── backend/    # Express API
├── infra/          # Server-Setup & Deployment
├── scripts/        # Hilfsskripte
└── memory/         # Projekt-spezifisches Wissen
```
